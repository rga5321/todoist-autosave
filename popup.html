⚡ todoist-autosave

How to use:
• Click the extension icon to save current URL
• Use keyboard shortcut (configurable in Firefox: about:addons → Manage Extension Shortcuts)

Getting your API key:
Go to Todoist → Settings → Integrations → Developer → API token

Finding Project ID:
Open a project in Todoist web, check the URL: todoist.com/app/project/ID
